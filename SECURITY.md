# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Если вы обнаружили уязвимость, пожалуйста, напишите на почту ernest779977@gmail.com.

Мы рассмотрим ваше сообщение в течение 48 часов и уведомим вас о решении. Если уязвимость подтвердится, мы выпустим обновление как можно скорее и сообщим вам о сроках. Вы также можете отправить сообщение через форму обратной связи в приложении (в профиле).
